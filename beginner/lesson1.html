Lesson 1: Introduction to C++ and Environment Setup

Introduction

C++ is a versatile programming language known for its performance, efficiency, and flexibility. It is widely used in developing software, games, and system-level applications. In this lesson, you will learn about the origins of C++, set up your development environment, and write your first C++ program.

Core Content

1) What is C++?

  • History: Developed by Bjarne Stroustrup at Bell Labs in 1985 as an extension of the C language. It was designed to provide object-oriented features while retaining C's low-level capabilities.
  • Applications: Used in software development (like operating systems, browsers, and games), embedded systems, and high-performance applications.
  • Key Features:
    • Object-Oriented: Supports classes and objects for structuring code.
    • High Performance: Efficient memory management and direct hardware access.
    • Portable: Can run on various platforms with minimal or no changes to the code.

2) Setting Up Your Development Environment

  • Choose an IDE (Integrated Development Environment):
    • Options: Visual Studio (Windows), Code::Blocks (cross-platform), VS Code (cross-platform).
    • Why use an IDE: Provides tools like code editors, debuggers, and compilers to write, debug, and run programs efficiently.
  • Installation Guide:
    • Visual Studio:
      • Download the installer from Visual Studio's official website
      • Run the installer, select "Desktop development with C++," and follow the on-screen instructions.
    • Code::Blocks:
      • Download the installer from Code::Block's official website
      • Install the software by following the setup wizard.
    • VS Code:
      • Download the installer from Visual Studio Code's official website
      • Install the "C/C++" extension from the Extensions Marketplace.
      • Configure the C++ compiler (e.g., GCC) according to your operating system.

Writing Your First C++ Program

  • You can start coding with your IDE or use Code Editor below

    • LET'S START WITH HELLO WORLD BY USING C++ 
    				 
					
					#include <iostream>  // Preprocessor directive to include input-output library
					using namespace std; // Allows the use of standard namespace

					int main() {
						cout << "Hello, World!" << endl; // Output "Hello, World!" to the console
						return 0; // End of program
					}

    Explanation of Each Component

    • #include <iostream> : Includes the library needed for input and output operations.
    • using namespace std; : Allows the use of names from the standard library
    • int main() { ... }: The main function where the program starts executing.
    • cout : Outputs text to the console.
    • endl : Inserts a newline character.
    • return 0; : Indicates the program ended successfully.
