Lesson 2: Variables and Data Types

Introduction

Variables are used to store data that your program can manipulate. Understanding variables and data types is crucial for any programming language. In this lesson, you will learn how to declare variables, understand different data types, and use them effectively in C++.

Core Content

1) What Are Variables?

  • Variables are containers for storing data values. They are used to label data with a descriptive name, making it easier to understand and manage.
  • Variables must be declared with a specific data type before use.
  • Data Types in C++
  • Primitive Data Types:
    • int: Stores integer values (whole numbers). Example: int age = 30;
    • float: Stores floating-point numbers (numbers with decimals). Example: float temperature = 36.6;
    • double: Stores double-precision floating-point numbers for more accuracy. Example: double pi = 3.14159;
    • char: Stores a single character. Example: char initial = 'A';
    • bool: Stores Boolean values ( true or false ). Example: bool isAdult = true ;
  • Example Declarations:
    • int score = 100;
    • float price = 19.99;
    • char grade = 'A';
    • bool isPassed = true;

  • Declaring and Initializing Variables
  • Declaration Syntax: dataType variableName;
    • int count; // Declaration
    • count = 10; // Initialization
  • Initialization: Assigning an initial value when declaring a variable.
    • int count = 10; // Declaration and initialization

  • Scope and Lifetime of Variables
  • Local Variables: Declared inside a function or block; accessible only within that function or block.
  • Global Variables: Declared outside any function; accessible throughout the program.
  • Example 
      • 							
					int globalVar = 5; // Global variable

					int main() {
						int localVar = 10; // Local variable
						cout << "Global Variable: " << globalVar << endl;
						cout << "Local Variable: " << localVar << endl;
						return 0;
					}
  • Simple Program to Demonstrate Variables: 
    • 						 

							#include  <iostream >
							using namespace std;

							int main() {
								int age = 30;
								float height = 5.9;
								char gender = 'M';
								bool isStudent = false;

								cout << "Age: " << age << endl;
								cout << "Height: " << height << " feet" << endl;
								cout << "Gender: " << gender << endl;
								cout << "Is Student: " << (isStudent ? "Yes" : "No") << endl;
								return 0;
							}
